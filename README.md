介绍:Remi是一个独立的Python GUI库,只有不到100K的源代码，不需要精通HTML知识就能轻松写出交互性强的网页GUI界面。这个项目小巧轻量，适合快速部署一些小型项目，也适合爱折腾的极客们使用。它可以轻松帮助我们搭建小型网页，帮助小白更好的理解前端知识。

这个仓库主要记录我学习remi的一些例程，具体学习过程请移步到我的Blog[四叶草の博客](http://www.wzcwzc.cool/)
